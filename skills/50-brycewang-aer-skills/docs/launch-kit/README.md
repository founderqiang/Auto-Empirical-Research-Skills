# Launch Kit（发布宣发材料包）

v1.2/v1.3 发布后的人工宣发动作与现成草稿。每个文件可直接复制修改后发布。

| 文件 | 用途 |
|---|---|
| `announcement-zh.md` | 中文平台草稿：知乎回答/文章、小红书、微信公众号 |
| `announcement-en.md` | 英文平台草稿：X/Twitter thread、Hacker News (Show HN)、Reddit |
| `awesome-list-submissions.md` | 向各 awesome 清单提交的 PR 文案 + Zenodo DOI 存档操作步骤 |

## 发布前 checklist

- [ ] 确认 main 上 CI 全绿、最新 release 已发布
- [ ] 数字与 `docs/quality-scorecard.md` 一致（demo 数、断言数、测试数会随版本变化）
- [ ] 所有链接指向 `main` 分支的稳定路径
- [ ] Zenodo 集成开启后，把 DOI 徽章回填进双语 README
