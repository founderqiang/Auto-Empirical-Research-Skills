# Chinese Data Catalog Quick Reference

## Source 1: 皮皮侠数据 (ppmandata.cn) — 1,599 datasets

### 企业创新与数字化 (260 datasets)
Key datasets: AI创新产出, 持续性创新投入, 企业创新质量, 数字供应链金融指数, AI技术创新, 漂AI指数, 知识多元化指数, 联合创新, 技术融合变量, 制造业-AI耦合专利, 数字政府建设
Coverage: mostly 2000-2025, firm-level (listed companies)

### ESG与绿色金融 (169 datasets)
Key datasets: ESG评分, 绿色补贴DID, 策略性环境信息披露, 气候风险指数, 绿色创新波动性, 碳排放, 环保关注度, 新能源宣传强度
Coverage: mostly 2010-2025, firm-level and province-level

### 区域与宏观 (144 datasets)
Key datasets: 夜间灯光, 城市通勤时耗, 城市网络关注度, 区域经济差距, 旧海关数据(1842-1949), 政府关注度, 公共服务注意力
Coverage: city/province level, various periods

### 政策DID (126 datasets)
Key datasets: 司法体制改革DID, 城市群政策DID, 海外并购DID, 直播电商DID, 固定资产加速折旧DID, 混合所有制改革DID, 数字治理DID, 服务贸易创新DID, 绿色补贴DID, 低空经济DID
Coverage: 2000-2025, firm-level and city-level panels

### 金融市场 (65 datasets)
Key datasets: 股价同步性, 与金融监管机构距离, 影子银行发展指标, 数字普惠金融, 风险承担水平, 绩效期望盈余, 管理层短视指数
Coverage: firm-level, mostly 2000-2025

### 文本与舆情 (42 datasets)
Key datasets: 国务院发布会文本, 百度搜索指数(银发经济/低空经济), 美国政府工作报告, 地方政府工作报告语调, 董事会公告文本, 经济政策不确定性感知
Coverage: various levels and periods

### 公司治理 (31 datasets)
Key datasets: 纵向共同机构投资者, 连锁董事网络, 高管数据, 独立董事网络, 实际控制人
Coverage: firm-level, mostly 2000-2025

### 劳动与人力资本 (25 datasets)
Key datasets: 受教育年限与学历结构, 劳动力技能结构指数, Glassdoor就业岗位
Coverage: province/firm level

### 供应链 (17 datasets)
Key datasets: 供应商与客户网络稳定性, 供应链风险感知, 供应链韧性, 供应链成本分摊, 供应链空间距离
Coverage: firm-level, mostly 2001-2024

## Source 2: 马克数据 (macrodatas.cn) — 377 datasets

### Key Categories
- 数字经济: 地级市数字经济指数(2011-2024), 数字化转型MD&A词频
- 环境: CO2排放(省/市/县, 1970-2024), 排污权交易DID
- 人口与社会: 人口空心化, 结婚离婚率, 文盲率, 平均预期寿命
- 产业: 产业结构高级化/合理化, 高新技术企业数据库
- 政策文本: 地级市政府工作报告(2002-2025), 北大法宝环保处罚
- 企业微观: 异地投资, 环保投资额, 创新困境, 供应链韧性, 碳排放量
- 消费: 地级市消费水平(2000-2024), 老龄消费需求
- 金融: 数字普惠金融, 耐心资本投资

### Common Data Sources Referenced
- CSMAR (国泰安): 中国上市公司财务、治理、交易数据
- Wind (万得): 金融终端数据
- CNRDS: 中国研究数据服务平台
- 北大数字金融研究中心: 数字普惠金融指数
- 国家统计局: 宏观经济数据
- 中国人民银行: 金融统计数据
