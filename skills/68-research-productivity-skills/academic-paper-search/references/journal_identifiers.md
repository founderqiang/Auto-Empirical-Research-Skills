## Usage Examples

### OpenAlex: Search within a specific journal
```bash
# Search AER for papers on "auction"
curl -s "https://api.openalex.org/works?filter=primary_location.source.id:S23254222,default.search:auction&per_page=10&mailto=you@university.edu"

# Search multiple top-5 journals
curl -s "https://api.openalex.org/works?filter=primary_location.source.id:S23254222|S95464858|S203860005|S95323914|S88935262,default.search:network+formation&per_page=25&mailto=you@university.edu"
```

### CrossRef: Search within a specific journal
```bash
# Search Econometrica by ISSN
curl -s "https://api.crossref.org/journals/0012-9682/works?query=identification&rows=10&mailto=you@university.edu"
```
