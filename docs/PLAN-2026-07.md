# AERS 2026-07 月度工作规划（目标：全网第一的自动实证研究仓库）

> 制定日期：2026-07-01。本计划以「可验证的严谨性」为核心差异化：AERS 不只是最大的 skill 集合，
> 而是唯一一个对方法族做**数值恢复基准（benchmark）+ 方法陷阱评测（eval）双重闭环**的实证研究仓库。
> 每周目标都有明确的完成判据（`make check` 全绿 + 生成物新鲜度检查通过）。

## 竞争定位（为什么这个计划能做到第一）

同类仓库（awesome 链接合集、单点 econometrics agent、论文写作 prompt 库）都缺少以下至少一项，而 AERS 四项全有并持续加深：

1. **全部 vendored**（1,145 个 skill 进仓库，不是外链），带许可证审计与安全扫描；
2. **机器可读 catalog + 溯源**（`catalog/*.json`），CI 强制新鲜度；
3. **严谨性双闭环**：eval-harness（会不会掉进方法陷阱）× benchmark（能不能算对数字）；
4. **整仓路由 Skill**：根 `SKILL.md` 让任何 agent IDE 一键导入并按需加载。

本月工作沿这四条护城河加深，并补齐「发布自动化 + 生态对比 + 增长面」。

## 第 1 周（07-01 ~ 07-07）：严谨性纵深 —— 方法族覆盖 11 → 13

- [ ] 新增 **CATE / 异质处理效应** 方法族（catalog 中 8 个 skill 支持）：
  - taxonomy 关键词规则（causal forest / meta-learner / CATE / heterogeneous effects / policy tree）；
  - eval 场景 `statspai-heterogeneous-effects`（诚实样本分割、重叠性、GATE/校准检验、子组多重检验）；
  - benchmark 任务 `cate-recovery`（确定性模拟：分层随机化 + 符号相反的子组效应；
    检验点：分层估计恢复各子组真值、朴素合并估计偏误被暴露、异质性差距被恢复、防伪造数字诚实校验）。
- [ ] 新增 **分位数处理效应 QTE** 方法族（2 个 skill 支持）：
  - taxonomy 关键词规则（quantile treatment effect / QTE / qreg / RIF / unconditional quantile）；
  - eval 场景 `statspai-quantile-effects`（QTE ≠ 分位点上的个体效应、秩不变假设、多分位点报告、自助法推断）;
  - benchmark 任务 `qte-recovery`（确定性配对设计：处理只抬升上尾；
    检验点：中位数 QTE=0、Q90 QTE=5、均值效应=1 全部恢复——「均值掩盖尾部」这一课被数值化）。
- [ ] `RIGOR_COVERAGE.md` 覆盖图 11 → 13 个方法族全闭环；eval/benchmark 数量下限棘轮同步上调。
- [ ] 单元测试覆盖两个新模拟库的真值不变量。

**完成判据**：`make check` 全绿；覆盖图脚注显示 13 个方法族 0 缺口。

## 跨周原则

- 一切生成物（catalog、覆盖图、release notes）必须有 `--check` 新鲜度模式并接入 `make validate`；
- 数量下限（scenario 数、auto-check 数、benchmark 任务数）只升不降（棘轮）；
- 新增内容一律先过 `make check` 再提交，PR 粒度按逻辑分块。
