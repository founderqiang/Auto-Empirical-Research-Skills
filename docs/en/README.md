## How to use this workflow

Each guide follows the same structure:

1. **Why this stage matters** — a one-paragraph motivation.
2. **Vendored skill collections** — pre-installed skills in the repo that are most useful at this stage. Click through to the folders to inspect them.
3. **External skill catalogue** — third-party skills you can install for additional capability.
4. **Workflow recommendations** — concrete, ordered steps for executing the stage.
5. **Practical advice** — pitfalls, gotchas, and trade-offs to keep in mind.

The terminology used across these guides mirrors the Chinese originals: methods
are referred to by their canonical names (DiD, IV, RDD, DML, PSM, SCM, ARDL,
GMM, etc.); econometric jargon such as "first-stage F", "parallel trends",
"honest-DID", or "Borusyak-Jaravel-Spiess imputation" is preserved in English.

## Translating back to Chinese

If you need the Chinese original, open the sibling file in the parent
`docs/` directory. Each Chinese file carries a banner at the top pointing
back to its English counterpart, e.g.:

> 🇬🇧 **English version available:** [docs/en/01-research-topic-and-design.md](01-research-topic-and-design.md)
